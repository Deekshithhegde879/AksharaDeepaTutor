PROJECT REQUIREMENTS DOCUMENT 
Akshara-Deepa Tutor — Smart Learning Companion
Android Application with GenAI Integration   
Deekshith 
1. Introduction 
Akshara-Deepa Tutor is an Android-based educational application designed to support self-learning among 
10th-grade (SSLC) students, especially in rural areas. Many students rely only on textbooks without a 
structured way to monitor their understanding or progress. 
This application transforms the syllabus into an interactive learning journey by tracking chapter completion, 
conducting self-assessment quizzes, and identifying weak areas. With the help of intelligent logic and GenAI
based insights, the app guides students to improve their performance through consistent practice and progress 
tracking. 
2. Problem Statement 
Students in rural regions often lack proper guidance during self-study. While they have access to textbooks, 
they do not have tools to evaluate their understanding or track their learning progress effectively. 
As a result, knowledge gaps remain unnoticed until final examinations, which negatively impacts 
performance and confidence. There is a need for a system that continuously monitors progress and highlights 
weak subjects early. 
3. Objectives 
• To provide a structured self-learning platform for SSLC students  
• To track chapter-wise progress across subjects  
• To conduct regular self-assessment quizzes  
• To identify weak areas and improve learning outcomes  
• To promote independent and consistent study habits  
• To support offline learning for accessibility  
4. Scope 
In Scope 
• Syllabus tracking for multiple subjects  
• Chapter-wise completion marking  
• Daily quiz system with simulated questions  
• Strength analysis using performance data  
• Daily study reminders  
• Offline functionality after initial data load  
5. Functional Requirements 
1. The system shall display a structured syllabus for subjects like Science, Math, and Social Studies  
2. The system shall allow users to mark chapters as completed  
3. The system shall provide daily quizzes with multiple questions  
4. The system shall evaluate quiz performance and update scores  
5. The system shall generate a visual “Strength Map” based on performance  
6. The system shall provide reminders for daily study goals  
7. The system shall allow users to review quiz answers  
8. The system shall function offline after loading initial data  
6. Non-Functional Requirements 
• Performance: App should respond instantly to progress updates  
• Reliability: Data should be stored locally and remain consistent  
• Usability: Interface should be simple and student-friendly  
• Accessibility: Must work without internet after setup  
• Scalability: Should support additional subjects and content  
• Compatibility: Android version 9.0 and above  
7. User Requirements 
Student Requirements: 
• Easy tracking of syllabus completion  
• Regular self-assessment through quizzes  
• Clear identification of weak subjects  
• Motivation through daily goals and reminders  
9. System Requirements 
Hardware: 
• Android smartphone  
Software: 
• Android OS (version 9 or above)  
• Local database (for offline content)  
• GenAI integration (optional enhancement)  
10. Data Requirements 
Syllabus Data: 
• Subject Name  
• Chapter List  
• Completion Status  
Quiz Data: 
• Question ID  
• Chapter Reference  
• Question & Options  
• Correct Answer  
Performance Data: 
• User Score  
• Chapter-wise Performance  
• Strength Metrics  
11. Constraints 
• Limited development time  
• Use of simulated/mock data for quizzes  
• No continuous internet dependency  
• Limited storage for offline database  
12. Expected Outcome 
Akshara-Deepa Tutor is expected to improve self-learning efficiency among students by providing a 
structured and trackable study system. It will help students identify their weak areas early, improve academic 
performance, and build confidence. The application also promotes digital learning practices and supports 
education in areas with limited resources.
